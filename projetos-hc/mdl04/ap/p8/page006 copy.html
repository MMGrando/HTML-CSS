Controles sem tag 'input'

Até o momento, foram utilizados apenas controles com a tag 'input'. Será utilizado agora alguns controles que não usam essa tag: select, datalist e textarea.

Os três são elementos de formulário que oferecem meios de entrada e seleção de dados com ofuncionalidades bem distintas.

Controles:

    ¬ Select:

  • type="select": cria um controle de lista suspensa que permite ao usuário selecionar uma ou mais opções de um conjunto predefinido;
  • 'option': define as opções disponíveis no menu suspenso;
  • 'optgroup': agrupa e classifica os 'options' (usar label);
  • 'multiple': permite selecionar múltiplas opções ao mesmo tempo.

    • ¬ Datalist:

  • type="datalist": fornece uma lista de sugestões para um campo de entrada de texto, aparecendo quando o usuário começa a digitar;
  • 'option': define as sugestões disponíveis ;
  • 'optgroup': agrupa e classifica os 'options' (usar label);
  • 'list': associa o campo de entrada de texto à lista de sugestões definida com a tag 'datalist'.

    • ¬ Textarea:

  • type="textarea": cria um campo de texto multilinha que permite a entrada de grandes volumes de texto;
  • 'rows': define o número de linhas visíveis no campo de texto;
  • 'cols': define o número de colunas visíveis no campo de texto;
  • 'placeholder': fornece um texto de exemplo ou dica sobre o que deve ser inserido no campo.

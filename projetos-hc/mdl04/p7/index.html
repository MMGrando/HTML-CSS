Formulário com label

No texto de solitação de informação coloca-se um 'label', relacionando assim esse 'label' ao 'input' em sequência.

Além disso, o texto em si passa a ser clicável e ativando o input (muito útil para uso mobile).

Métodos 'GET' e 'POST' para formulários

O método 'GET' é o método padrão, mas com ele os dados do formulário são inseridos na URL. Já o método 'POST' esconde os dados da URL, ainda que os dados sejam facilmente encontrados ao inspecionar no network o cadastro feito.

Para usá-los basta colocar o parâmetro 'method=" " ' em 'form', e preencher com o método desejado.

Quando usar 'GET' ou 'POST'?

  • Método Get
  • Dados não sensíveis
  • Nomes, peso, altura
  • Dados de "procurar"
  • Marca de carro/roupa, etc
  • Ou seja, tudo que não sejam dados sensíveis e que, o formulário como um todo, NÃO EXCEDA 3 mil bytes (3 mil caracteres).
  • Método Post
  • Dados sensíveis
  • Cartão, senha
  • Endereço, CEP
  • Envio de arquivos
  • Ou seja, dados que necessitam um mínimo de sigilo ou que,o fomulário como um todo, EXCEDA 3 mil bytes (3 mil caracteres).

Conhecendo Media Query

Conhecendo um pouco sobre 'Media Query', suas categorias ('Media Types'), suas propriedades ('Media Features') e como elas se relacionam e podem ser aplicadas.

Media Query: uma técnica de CSS que permite aplicar estilos específicos com base na tela de cada dispositivo que está exibindo o conteúdo.

Media Types: são categorias gerais que permitem escolher o tipo de mídia para o qual o estilo será aplicado. Exemplos incluem tela (screen), impressão (print), entre outros.

Media Features: são propriedades ou configurações específicas que definem as condições necessárias para que o estilo CSS seja aplicado.

Definições:

    ¬ Media Queries

  • No HTML: criando uma opção de 'link' para cada media type e feature desejados;
  • No CSS: utilizando @media dentro do próprio arquivo CSS para cada media type e feature desejados;

    ¬ Media Types

  • 'all': estilo aplicado a todos os dispositivos;
  • 'print': estilo aplicado apenas em documentos para impressão;
  • 'screen': estilo aplicado em telas em geral;
  • 'speech': estilo aplicado para sintetizadores de voz;
  • 'projection': estilo aplicado para projeções;

    ¬ Media Features

  • OBS: Sempre usar a definição entre parênteses (features);
  • 'width/height': define a aplicação pela largura e/ou altura da viewport;
  • 'device-width/height': define a aplicação pela largura ou/e altura da tela do dispositivo;
  • 'orientation': define a aplicação pela orientação da tela (portrait ou landscape), conhecido como "em pé ou deitado";
  • 'resolution': define a aplicação pela resolução da tela (dpi ou dpcm).

Um pouco mais sobre:

De forma geral, uma 'media query' utiliza 'media types' e/ou 'media features' para definir as condições necessárias para que estilos específicos sejam aplicados.

Basicamente, uma 'media query' é uma regra que avalia um ou mais 'media types' e/ou 'media features' para aplicar estilos CSS conforme os critérios definidos.

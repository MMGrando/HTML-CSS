Conhecendo mais sobre formulários

Importante ressaltar que alguns atributos e controles podem não funcionar dependendo do navegador. Portanto, deve-se sempre verificar os principais navegadores antes de definir o uso de cada ferramenta.

Nem todos os atributos funcionando em todos os controles. Por exemplo, os atributos max/min definem o número máximo e número mínimo aceito pelo input, ou seja, são atributos para controles numéricos. Já minlenght/maxlenght são número totais de caracteres aceitos, funcionam num geral.

Mais alguns controles:

  • type="number": cria um campo de entrada numérica onde o usuário pode inserir números;
  • type="month": cria um campo para selecionar um mês e um ano.;
  • type="date": cria um campo para selecionar uma data completa (dia, mês e ano);
  • type="time": cria um campo para selecionar uma hora e minutos.

Mais alguns atributos:

  • value="": Pré estabelece um valor inicial para o input;
  • min="": define o menor número aceito pelo input;
  • max="": define o maior número aceito pelo input;
  • step="": define o incremento entre os valores no input (1.0, 1.1, 1.2.. ou 1.0, 1.5, 2.0..).

Usando esses controles/atributos

'input type="month"'

'input type="date"'

'input type="time"'

OBS: O valor de 'value=""', dentro de 'month' e 'date', foi atribuído manualmente. Mas pode ser usado javascript para que esse processo seja feito sempre, atualizando com a data/mês atual.

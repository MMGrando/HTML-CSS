Controles visuais

Inputs 'type="color"', 'type="range"' e 'type="file"' são controles de formulário que oferecem interfaces e interativas para seleção e ajuste visuais.

Color: Seletor visual de cores.
Range: Controle deslizante entre intervalos de valores.
File: Permite selecionar e faezr upload de arquivos.

Importante conferir a compatibilidade dos navegadores aos controles usados. Observar como esses inputs se comportam, pelo menos, nos principais navegadores do mercado.

Controles:

    ¬ Color:

  • type="color": cria um controle que permite ao usuário selecionar uma cor;
  • 'value': define uma cor pré definida;

    • ¬ Range:

  • type="range": ria um controle deslizante que permite ao usuário selecionar um valor dentro de um intervalo especificado;
  • 'min': define o valor mínimo do intervalo;
  • 'max': define o valor máximo do intervalo;
  • 'step': define o incremento dos valores no intervalo.

    • ¬ File:

  • type="file": cria um controle que permite ao usuário selecionar um ou mais arquivos de seu dispositivo para upload;
  • 'accept': define os tipos de arquivo que o controle pode aceitar (por exemplo, accept="image/*" para apenas imagens);
  • 'multiple': permite selecionar múltiplos arquivos.

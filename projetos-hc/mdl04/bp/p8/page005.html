Controles de opções e seus 'values'

Os 'inputs' checkbox e radio são usados para criar opções de seleção em formulários, permitindo que o usuário possa interagir escolhendo uma ou mais oções. Esses controles oferecem diferentes formas de coleta de dados do usuário.

Checkbox: Seleção múltipla.
Radio: Seleção Única.

O atributo 'value' desses 'inputs' define o valor enviado ao servidor quando o formulário for finalizado ('submetido'). É crucial para identificação e processamento das escolhas do usuário.

Por exemplo, se o usuário marcar 'futebol' mas o atributo value estiver definido como 'vôlei', o servidor receberá 'vôlei'. Portanto, é essencial revisar e garantir que o value esteja corretamente configurado para refletir as opções selecionadas.

Controles:

    ¬ checkbox:

  • type="checkbox": cria uma caixa de seleção que permite marcar múltiplas opções simultaneamente;
  • 'value': define o valor enviado ao selecionar a caixa;
  • 'name': agrupa caixas de seleção para processamento no servidor;
  • 'checked': determina a caixa pré selecionada inicialmente.

    • ¬ radio:

  • type="radio": cria opções de seleção onde apenas uma opção pode ser marcada por grupo;
  • 'value': define o valor enviado ao selecionar o botão;
  • 'name': agrupa as opções de seleção para permitir seleção única;
  • 'checked': determina uma opção pré selecionada inicialmente.

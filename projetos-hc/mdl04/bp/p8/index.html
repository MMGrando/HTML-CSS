Construindo um formulário

Ao decorrer da página farei alguns passos que compõem um formulário de cadastro 'login e senha', juntamente com o uso de alguns importantes atributos para um funcionamento certo desses controles.

Os 'input' são controles com suas respectivas funcionalidades.
Os 'atributos' são as configurações desses controles.

Alguns controles:

  • type="text": cria um campo de texto para inserção de dados;
  • type="password": cria um campo de 'texto oculto' para inserção de senhas;
  • type="submit": cria um botão para envio de dados do formulário;
  • type="reset": cria um botão para redefinir os dados do formulário.

Alguns atributos:

  • required: torna o campo de preenchimento obrigatório;
  • placeholder="": texto temporário que serve como dica do valor esperado;
  • minlength="": define o número mínimo de caracteres necessário;
  • maxlength="": define o número máximo de caracteres permitido;
  • size="": largura do campo de preenchimento (em nº de caracteres).

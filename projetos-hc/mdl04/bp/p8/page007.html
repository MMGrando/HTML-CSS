Outputs - apresentação de dados

Os elementos 'output' são usados para mostrar resultados ou valores calculados para os usuários. Eles exibem dados dinamicamente com base nas interações com outros controles de entrada, como campos de texto, seletores e sliders.

O atributo 'oninput' é um evento que executa uma função JavaScript sempre que o valor de um campo de entrada muda. Quando usado em conjunto com 'output', ele permite que a saída seja atualizada em tempo real, proporcionando feedback imediato ao usuário sobre suas ações.

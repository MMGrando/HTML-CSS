Tabalhando com Listas

Ver sobre listas mistas e de definição!

Existem três tipos de listas:

Lista 'ordenada':
Lista de itens feito sob uma ordem numerada ou alfabética. Cada item recebe um número ou letra que destaca sua ordem.
Lista 'não ordenada'':
Itens da lista não possuem nenhuma ordem definida. Cada item recebe um símbolo que o espefifica.
Lista 'de definição':
Cada item listado pode receber uma definição (ou descrição) de seu significado.

Sobre listas ordenadas e não ordendas:

    Lista ordenada 'ol'>'li'

  1. Manteiga
  2. Tomate
  3. Carne
  4. arroz

    Lista não ordenada 'ul'>'li'

  • Manteiga
  • Tomate
  • Carne
  • arroz

Regras úteis CSS

list-style-position:
define a formatação dos símbolos (outside = padrão/fora da formatação, inside = dentro da formatação);
'list-style-type':
define o simbolo usado para cada item listado. (square = quadrado, disc = bolinha, circle = círculo oco, etc)
